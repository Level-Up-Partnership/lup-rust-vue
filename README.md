# rust-app

# Getting started:

# Backend:

Rebuild on every save:

`cargo watch -x run`

To build rust package manager:

`cargo build`

To run:

`cargo run`

To run tests:

`cargo test`

# Frontend (webui):

Change directory into frontend folder.
`cd webui`

## Project setup

```
npm install
```
